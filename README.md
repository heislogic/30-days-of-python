# 🐍 Desafio Python 30 Dias: Do Zero ao Projeto Final

Este repositório contém minha jornada de 30 dias aprendendo Python do absoluto zero. O objetivo é desenvolver a lógica de programação através de scripts úteis para o cotidiano, escalando a complexidade a cada bloco.

> "O código não é redação, é instrução."

---

## 🚀 Status do Desafio
| Bloco | Status | Descrição |
| :--- | :---: | :--- |
| **01: Utilidade Pessoal** | 🟡 20% | Scripts para o dia a dia e fundamentos. |
| **02: Automação** | ⚪ 0% | Scripts que interagem com o sistema e web. |
| **03: Diferencial** | ⚪ 0% | Projetos de portfólio e integração. |
